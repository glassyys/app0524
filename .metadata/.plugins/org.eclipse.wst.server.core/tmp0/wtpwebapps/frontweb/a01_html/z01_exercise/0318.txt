
오늘 제공한 HTML 코드의 특징들을 활용하여 실습 연습예제를 만들면 다음과 같은 3가지 문제를 제안할 수 있습니다.

연습예제 1: 자기소개 페이지 만들기(0318_01.html)
<title> 태그를 사용하여 브라우저 탭에 표시될 페이지 제목을 "나의 소개"로 설정합니다.
<h1> 태그를 사용하여 페이지의 주제인 "자기소개"를 큰 제목으로 표시합니다.
<p> 태그를 사용하여 여러 단락으로 나누어 이름, 취미, 꿈 등에 대해 간단히 소개합니다.
<hr> 태그를 사용하여 각 섹션을 구분합니다.
<b>나 <strong> 태그를 사용하여 중요한 부분을 강조합니다.

연습예제 2: 간단한 수학 문제 페이지 만들기(0318_02.html)
페이지의 제목을 "간단한 수학 문제"로 설정합니다.
수학 문제들을 <h3> 태그로 제목을 붙여 나열합니다. 예: "문제 1", "문제 2".
각 문제에 대한 설명을 <p> 태그로 작성합니다.
수학 기호를 사용하여 문제를 표시합니다. 예: "3 × 4 = ?", "12 ÷ 4 = ?".
<pre> 태그를 사용하여 복잡한 수식을 정확하게 표현합니다.

연습예제 3: HTML 텍스트 스타일링 연습(0318_03.html)
페이지 제목을 "텍스트 스타일링 연습"으로 설정합니다.
<h2> 태그로 "텍스트 꾸미기" 섹션을 시작합니다.
<p> 태그 내에 다양한 텍스트 스타일링을 실습합니다. 예를 들어, <b>, <i>, <mark>, <del>, <ins>, <sup>, <sub> 등을 사용하여 문장을 꾸며 봅니다.
각 스타일을 적용한 예시를 <br> 태그로 줄바꿈하여 나열합니다.
<hr> 태그로 섹션의 시작과 끝을 명확히 구분합니다.
