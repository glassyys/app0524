### 문제 1: 로그인 폼 만들기
웹페이지에 로그인 폼을 만드세요. 사용자에게 사용자명과 비밀번호를 입력받아야 합니다. 사용자명은 텍스트 입력 타입을, 비밀번호는 비밀번호 입력 타입을 사용하세요. 각 입력 필드는 적절한 placeholder를 포함해야 합니다.

### 문제 2: 사용자 피드백 양식
웹사이트 방문자로부터 피드백을 수집하는 다중행 입력 양식을 만드세요. 이 양식은 사용자의 이름, 이메일 주소, 그리고 피드백 메시지를 포함해야 합니다. 이름과 이메일은 단일 행 입력 필드를 사용하고, 피드백 메시지는 다중행 입력 필드를 사용하세요. 모든 필드는 필수로 입력해야 합니다.

### 문제 3: 음식 주문 폼
음식 주문 폼을 만드세요. 사용자는 피자, 파스타, 샐러드 중에서 하나 이상 선택할 수 있어야 합니다. 각 선택지는 체크박스를 사용하여 표현하세요. 또한, 사용자가 선호하는 음료를 텍스트 입력과 datalist를 사용하여 선택할 수 있도록 하세요. 음료 선택지로는 "콜라", "사이다", "물"이 있습니다.

### 문제 4: 회원 가입 양식
사용자로부터 회원 가입을 위한 정보를 수집하는 폼을 만드세요. 필요한 정보는 사용자의 이름, 이메일 주소, 비밀번호, 성별(남성, 여성 선택), 그리고 국가입니다. 국가 선택은 텍스트 입력과 datalist를 사용하여 제공되는 목록 중에서 선택하거나 직접 입력할 수 있도록 하세요. 국가 선택지로는 "한국", "미국", "일본"이 있습니다. 모든 필드는 필수로 입력해야 합니다.

### 문제 5: 설문조사 양식
새로운 제품에 대한 고객의 선호도를 조사하는 설문조사 양식을 만드세요. 설문조사는 사용자의 연령대(20대, 30대, 40대 선택), 선호하는 제품 색상(단일 선택), 그리고 제품에 대한 추가 의견을 수집해야 합니다. 연령대와 선호하는 제품 색상은 라디오 버튼을 사용하여 선택하게 하고, 추가 의견은 다중행 텍스트 입력을 사용하여 수집하세요.
