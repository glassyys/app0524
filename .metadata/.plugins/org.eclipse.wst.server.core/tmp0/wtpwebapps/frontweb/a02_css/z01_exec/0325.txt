### 실습 예제

#### 예제 1: 인라인 스타일 적용
- **목표**: HTML 요소에 직접 스타일을 적용해보세요.
- **과제**: `<p>` 태그의 텍스트에 인라인 스타일을 사용하여 색상을 'tomato'로, 폰트 크기를 '18px'로 설정하세요.

#### 예제 2: TAG, ID 선택자 활용
- **목표**: TAG, ID 선택자를 통한 CSS 적용 방법 이해하기
- **과제**: HTML 요소에 TAG, ID를 부여하고, 해당 TAG, ID를 가진 요소의 배경색을 'orange'/'lightblue'로 설정하세요.

#### 예제 3: 상속 이해하기
- **목표**: CSS의 상속 개념 이해하기
- **과제**: `<div>` 태그에 폰트 색상을 'darkgreen'으로 설정하고, 내부에 있는 `<span>` 요소에 대해서는 별도의 색상 설정 없이 상속받은 스타일을 관찰하세요.

#### 예제 4: 외부 스타일 시트 적용
- **목표**: 별도의 CSS 파일을 HTML 문서에 적용하는 방법 학습하기
- **과제**: 외부 CSS 파일(`main.css`)을 만들고, `<body>` 태그의 배경색을 설정한 후, HTML 문서에 이 파일을 링크하세요.
