#### 과제 1: CSS의 우선순위 규칙
- **주제**: CSS에서 인라인 스타일, ID 선택자, 태그 선택자 간의 우선순위는 어떻게 결정되나요? 
해당 규칙이 프로젝트에서 어떻게 활용될 수 있는지 팀원들과 논의해보세요.
답변)
인라인 스타일 최우선순위
그 다음 ID 선택자 적용되고
태그 선택자가 맨 나중에 적용됩니다.

#### 과제 2: CSS의 최신 기능 탐색
- **주제**: 공통 css 선언 방식이 웹 개발에 메뉴 설정시 어떤 영향을 미치는지 살펴보고,
   공통 css를 사용하느냐 개발 화면에서 css 사용하느냐 장단점을 토론하고, 
   실제 프로젝트에서 어떻게 활용할 수 있을지 고민해보세요.

답변)
공통 css 선언하면 모든 html페이지에서 공통기준으로 자동으로 적용됩니다.
장점:
표준폰트나 색상을 변경하고 싶을때 css파일내용만 변경하면 전체가 일괄 적용된다.
단점:
각 페이지마다 원하는 포맷을 적용하려면 별도의 추가 코딩이 필요하다.
