### CSS 연습 문제

#### 문제 1: 조합 셀렉터 사용
- **과제**: `<ul>` 내부에 있는 `<a>` 태그에만 특별한 스타일을 적용하여, 색상을 'blue'로 처리

#### 문제 2: 자식 셀렉터 활용
- **과제**: `<div>`의 직계 자식인 `<p>` 태그의 텍스트 색상을 'dodgerblue'로 설정하세요.

#### 문제 3: 자손 셀렉터 활용
- **과제**: `<article>` 태그 내의 모든 `<em>` 태그의 텍스트를 기울임체로 변경하고, 색상을 'purple'로 설정하세요.

#### 문제 4: 가상 클래스 `:first-letter` 활용
- **과제**: 모든 `<p>` 태그의 첫 글자의 폰트 크기를 '36px'로, 색상을 'red'로 설정하세요.

#### 문제 5: 구조적 가상 클래스 `:nth-child()` 활용
- **과제**: `<li>` 태그 중 홀수 번째 태그의 배경색을 'lightgrey'로 설정하세요.

#### 문제 6: 가상 클래스와 구조적 가상 클래스 복합 활용
과제: 모든 <p> 태그 중에서 첫 번째 <p> 태그만을 선택하여 배경색을 'lightblue'로 설정하고, 그 내부의 첫 번째 단어만 폰트 크기를 '20px', 색상을 'navy'로 변경하세요.
