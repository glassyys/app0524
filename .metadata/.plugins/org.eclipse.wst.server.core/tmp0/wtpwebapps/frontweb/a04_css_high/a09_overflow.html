overflow 프로퍼티

overflow에 hidden값을 적용하면 박스을 넘어가는 내용이 잘려 보이지 않습니다.

overflow에 visible값을 적용하면 콘텐츠가 박스를 넘어가도록 출력됩니다.

overflow에 scroll값을 적용하면 박스에 스크롤바를 붙여 출력합니다.
