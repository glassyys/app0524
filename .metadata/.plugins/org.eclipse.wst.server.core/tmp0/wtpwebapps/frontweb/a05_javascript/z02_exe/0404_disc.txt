답변:
형변환이란 데이터타입을 다른 데이터타입으로 변경하는 것
예)
1) 문자열 변수를 숫자로 변환하여 계산하기 위해서 사용

3) prompt( )의 경우 전달받은 값의 자료형과 관계없이 항상 문자열로 자동 변환된다.

주제2. 가독성을 좋게 하는 코드란 어떻게 하는 것을 말하는지?
효율적인 코드 방법에 대하여 토론 해보세요.
답변:
1) 코드 정리 정돈이 잘 되어있는 코드를 말한다
2) 의미있는 변수를 사용하여 변수가 어느 부분에 속해 있는지 쉽게 판단할 수 있는 코드를 말한다.
  -중의적으로 해석되는 단어를 피하고 구체적인 단어 사용
3) 코드에 대한 설명, 언제 코드를 작성하고 수정했는지 표기 등과 같은 주석처리를 효율적으로 하여 이해하기 쉽게 만든 코드를 말한다.
즉, 코드를 만든 사람뿐만 아니라 다른 사람이 보았을 때에도 이해하기 쉽게 정히된 코드가 가독성 있는 코드라고 할 수 있다.

2-1) 변수를 의미 있는 변수명을 처리하는 것으로 빠르게 프로그램을 해석
