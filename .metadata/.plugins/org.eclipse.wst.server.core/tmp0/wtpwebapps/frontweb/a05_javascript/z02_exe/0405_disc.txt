2. 논리연산자 `&&`, `||`, `!`를 사용하는 다양한 실제 예시를 찾아서 공유하세요.
각 연산자가 프로그램의 어떤 부분에서 유용하게 사용될 수 있는지 논의하세요.
답변:
1) 영화관람시 청소년 입장여부 판단할때
2) 할인을 적용할 때 기준 판단
3) 연산자별 예시
&& : 투표참여자 신분확인 시 사용, 년도, 월, 일까지 조건을 만족하는 사람만이 투표가능
|| : 기차에 탈 수 있는 사람(온라인으로 예매 || 현장구매)
! : 신검통지서 받은 경우를 구분(여자가 아닌경우 모두 해당) 

3. 증감 연산자 `++`, `--`의 전위(prefix) 표기법과 후위(postfix) 표기법의 차이에 대해 논의하세요.
각각의 사용 예를 코드와 함께 제시하세요.
답변:
증감연산자 기호을 변수 앞이냐 뒤에 표시하는냐의 차이입니다.
전위의 경우는 실행하면 적용이 되어서 출력되고
후위의 경우는 기존값이 출력되고 다음연산에서 증감이 적용된다.
	var bdObj = document.querySelector("body") //css선택하는 형식과 동일
	var num01 = 10; //초기화
	bdObj.innerHTML += "num01++:"+(num01++)+"<br>" //출력후 값을 1씩증가   :후위(postfix)
	bdObj.innerHTML += "num01:"+num01+"<br>"       
	bdObj.innerHTML += "++num01:"+(++num01)+"<br>" //먼저 증가하고 1씩증가 :전위(prefix)
	bdObj.innerHTML += "num01--:"+(num01--)+"<br>" //출력후 값을 1씩감소   :후위(postfix)
	bdObj.innerHTML += "num01:"+num01+"<br>"
	bdObj.innerHTML += "--num01:"+(--num01)+"<br>" //먼저 감소하고 1씩감소 :전위(prefix)

4. 복합 대입 연산자를 사용하는 경우와 그렇지 않은 경우의 코드를 비교하여,
복합 대입 연산자의 장점에 대해 논의하세요.
답변:
1) 복합 대입 연산자를 사용하는 경우:
int x = 10;
x += 5; 

2) 그렇지 않은 경우:
int x = 10;
x = x + 5; 
복합 대입 연산자의 장점은 다음과 같습니다:

a) 코드가 간결해서 가독성이 향상됩니다.
b) 실수를 줄일 수 있음
c) 컴파일러에 따라 성능 향상
d) 유지보수가 용이하다.
f) 반복적인 작업을 줄일 수 있습니다.
