QR코드 로그인 방법

  • ① [QR코드 생성하기] 버튼을 선택해 주세요.
  • ② 스마트기기 카메라 앱으로 QR코드를 촬영해 주세요.
  • ③ KB스타뱅킹 앱 내 확인코드와 PC화면 내 확인코드가 동일한지 확인해 주세요.
  • ④ 확인코드가 동일하다면 KB스타뱅킹 앱 내에 간편비밀번호를 입력해 주세요.
